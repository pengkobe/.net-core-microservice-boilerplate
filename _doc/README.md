## 效率

### 优点 

- 早期沟通成本较大，随着项目复杂度的提升，效率不会有较大的下降，与单体架构形成了鲜明对比
- 需求响应速度加快，适应敏捷开发，单体架构还必须懂整个架构逻辑才行
- 升级效率快，影响小

### 缺点

- 部署和运维难度加大，但可以利用 DevOps 等方式将运维工作自动化
